Chat + Tools + Integrated.

Your data, tools, and approvals wrap around a single conversation. Nothing feels bolted on because nothing is.

What's inside

Batteries, included.

Everything you need to build an agent-driven workspace.

Multi-agent
Create and manage multiple agents, each with its own name, personality, and knowledge base.
Multi-provider AI
Anthropic, OpenAI, Gemini, Together, or Ollama. Switch models without re-wiring anything.
Knowledge
Feed an agent your docs, notes, or scraped pages. It cites what it used, you see every source.
Built-in CRM
Contacts, companies, deals, tasks — the same store your agents read from. Not a bolt-on.
Tools & integrations
Gmail, Calendar, QuickBooks, WhatsApp, Telegram. Agents use them; you approve what matters.
Local-first
SQLite on disk. Your data never leaves your machine unless you point it somewhere else.
Brandable
Your logo, your company name, your accent color. Make it yours in a minute.
Deploy anywhere
One-click to Railway, or run it on a spare Raspberry Pi. Anywhere Python runs, Chatty runs.

Get started

Up and running in 5 minutes.

Three lines. The launcher sets up a virtualenv, installs deps, and starts the server.

$ git clone https://github.com/WWilson1017/chatty.git
$ cd chatty
$ python run.py
→ Chatty is running at http://localhost:8000
View on GitHub One-click deploy to Railway
Requires Python 3.10+ and Node 18+
